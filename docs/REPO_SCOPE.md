# Public Repo Scope

This repository is a curated public export of the Chopping Block beta release Unity project.

## Included

- `Assets/Scenes`
- `Assets/Scripts`
- `Assets/Sprites`
- `Assets/Audio`
- `Assets/Fonts`
- `Assets/Prefabs`
- `Assets/Resources`
- `Assets/Animations`
- `Assets/Materials`
- `Assets/Settings`
- Required root Unity asset/config files
- `Packages/manifest.json`
- `Packages/packages-lock.json`
- `ProjectSettings` files needed to open the project

## Excluded

- `Library`
- `Temp`
- `Logs`
- `obj`
- `UserSettings`
- `.vs`
- `.plastic`
- `.git`
- Recovery files
- Deprecated scene backups
- Unneeded third-party demo/example content
- Local private metadata files such as `*.private.0`

## Why The Repo Is Curated

The goal of this repo is to present the beta release application work, contributor story, and Unity project structure without publishing generated artifacts, private workspace metadata, or unnecessary baggage that does not help employers evaluate the team's work.
